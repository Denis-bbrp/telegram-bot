Инструкция по запуску Telegram-бота

1. Установи Python 3.8+ и pip
2. Установи зависимости:
   pip install -r requirements.txt

3. Укажи правильный ADMIN_GROUP_ID в config.py (узнать можно, добавив бота в группу и поймав сообщение)

4. Запусти файл:
   python bot.py

5. Готово! Бот работает. Проверь в Telegram.
